# 🐥 La Patoteca

Tu biblioteca personal de recomendaciones: pelis, series, libros, anime, docus y cómics.

## Cómo subir a GitHub Pages

1. Crea un repositorio en GitHub llamado `patoteca`
2. Sube todos estos archivos (`index.html`, `manifest.json`, `_config.yml`, `icon-192.png`, `icon-512.png`)
3. Ve a **Settings → Pages → Source: main branch / root**
4. Tu app estará en `https://olgaberrios.github.io/patoteca`

## Iconos necesarios

Necesitas dos imágenes del icono del pato:
- `icon-192.png` → 192×192 px
- `icon-512.png` → 512×512 px

Puedes usar el icono original redimensionado.

## Sincronización con GitHub Gist

1. Ve a https://github.com/settings/tokens
2. Crea un **Classic token** con permiso `gist`
3. Abre la app → ⚙️ Ajustes → pega el token → Guardar
4. La primera vez se crea el Gist automáticamente
5. En otros dispositivos: usa el mismo token + el Gist ID que aparece en ajustes

## Instalar como app en el móvil

- **iOS (Safari):** Abre la URL → botón compartir → "Añadir a pantalla de inicio"
- **Android (Chrome):** Abre la URL → menú → "Instalar app" o "Añadir a pantalla de inicio"
